# PsiKit

PsiKit is a logging and replay framework for FTC inspired by [AdvantageKit](https://docs.advantagekit.org). Data can be live streamed in a format compatible with [AdvantageScope](https://docs.advantagescope.org).

### Current Features

- Live streaming via the RLOG server protocol
- Compatible with [AdvantageScope](https://docs.advantagescope.org) for data visualization

### Coming Soon

- Live tuning of variables
- Writing log files compatible with AdvantageScope
- Log replay, allowing you to log new data that you didn't record while the robot was running
- Kotlin tooling to take advantage of the language's expressiveness
